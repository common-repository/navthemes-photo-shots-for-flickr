== Description ==
A very lightweight plugin to display your flickr photos shots on your site.

== Installation ==
1. Unzip Plugin. 
2. Upload `folder` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Widgets you will be seeing new widget "Flickr Shots By NavThemes", add it to your desired sidebar.
5. Update settings of widget as per your need.


== Frequently Asked Questions ==
= Does a sidebar needed this widget to work ? =
A theme must have sidebar in order to use this plugin
